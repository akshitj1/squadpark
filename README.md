# Setup
Install go from: [official site](https://golang.org/doc/install)
```
git clone git@github.com:akshitj1/squadpark.git
cd squadpark
```

# Run
`go run sim.go input.txt`

# Run tests
`go test ./...`

## Assumptions
- Reg. No. will only be validated for length. not for format
- Reg. No. is unique
